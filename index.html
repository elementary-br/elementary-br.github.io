Lançamento: elementary OS Loki 0.4.1!

Estamos orgulhosos de anunciar o lançamento de uma atualização importante para o OS elementar Loki: versão 0.4.1. Esta versão traz uma nova pilha de habilitação de hardware, todas as atualizações para Loki até o momento, além de um novo AppCenter.

Atualizações de abril para o Loki

Caso você tenha perdido isso, enviamos uma grande correção para um problema que faz com que as atualizações não apareçam. Se você ainda não o fez, você vai querer ler nosso post sobre como aplicar a correção para que você possa desfrutar de todas as ótimas coisas que lançamos em abril!
